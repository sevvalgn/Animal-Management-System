# Animal Management System

A menu-driven C# console application that demonstrates Object-Oriented Programming concepts through an interactive animal management system.

## Project Background

This project originally started as a small Object-Oriented Programming assignment focused on inheritance and polymorphism.

I later expanded the original exercise into a menu-driven console application.

### Initial version:

Basic inheritance example
Animal, Cat and Fish classes

### Expanded version:

Menu system
Animal creation
Animal removal
Single animal movement
Input validation
Polymorphism through virtual methods

## Features

-Create different animal types

*Cat
*Dog
*Bird
*Fish

-Display all animals currently stored in the system

-Move a single selected animal

-Move all animals at once

-Remove animals from the collection

-Input validation using TryParse

-Unique behavior for each animal type through polymorphism

## Animal Types

-Cat

Move: Run
Special Action: Jump

-Dog

Move: Run
Special Action: Bark

-Bird

Move: Fly
Special Action: Fly Higher

-Fish

Move: Swim
Special Action: Dive
